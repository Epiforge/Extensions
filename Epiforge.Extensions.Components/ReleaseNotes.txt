Finalizer disposal can no longer be monitored with events. Finalized async disposable types no longer block the finalizer. DynamicAsyncDisposable and DynamicDisposable now properly inherit from DynamicPropertyChangeNotifier instead of PropertyChangeNotifier. In exception extensions, exception property getters which throw will now report the thrown exception, cycles are prevented when walking inner exceptions, and null stack frame arrays will no longer throw an NRE. Generic arithmetic delegate caching will no longer thrash. PropertyChangeNotifier and DynamicPropertyChangeNotifier now permit null or empty property names. In reflection extensions, GetImplementationEvents, GetImplementationMethods, and GetImplementationProperties are now O(1) instead of O(n) when deduplicating. AsyncSynchronizationContext no longer uses exceptions for control flow, will throw when Post or Send are called on a disposed context, and create true copies.
